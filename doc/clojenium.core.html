find-element
(find-element driver selector)(find-element driver by selector)
Takes a driver, a by and a selector, and returns
a WebElement. Default is by css e.g.
(find-element driver "#input") =>
(WebElement id="input")

(find-element driver :id "input") =>
 (WebElement id="input")
